Construção de classes c/bootstrap

 class="[container / img][row / col, padding e margin][display + variações][gaps][cores][outras classes bootstrap][custom-class]"
example: class="row py-3 my-3 d-flex flex-column justifi-content-center gap-3 bg-primary nav-link custom-class-forcss"
** se possível nomear as classes com contexto... exemplo: se é uma classe do footer, que aninha ícones sociais, nomear: footer-social-links-div
evitando usar nomes de classes já utilizados no bootstrap
